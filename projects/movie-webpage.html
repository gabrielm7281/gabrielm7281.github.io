Ink and Echoes

Me and another person created a fictional movie poster using basic HTML and CSS

Context of This Project

We had a fun project where we could design a movie poster just by using HTML and CSS. It needed styling, especially from the blank sheet that me and Julien started from, which allowed us to practice combining these two major elements in Web Design. We decided to make a Horror movie called Inks and Echoes, which is mostly about a class on a trip, trying to stop a curse from escaping a possesed book.



What Was Required?

This project needed simple HTML and CSS to work as theres not much to look at or discover.

The biggest and most noticeable things that we needed were fonts, inputs (which are at the bottom), line and word spacing, as well as using margins and borders to put things in position.

Some other minor things:

  • Divs and Spans
  • Multiple Local Pictures
  • Other Characters


Challenges and Takeaways

I assigned myself to the CSS role so I did most of the colors, fonts, and borders. I think the biggest issue was trying to get the images to work properly. At the start I wanted to have background images for some parts of the poster, but they would only fill a certain section such as inside the border at the top. I did eventually get the code to work for me, but I didnt like how it looked compared to how the page was turning out. I decided to scrap the idea and move all the pictures to the places you see them in now. It was a lot simpler than trying to force it into a place that might not properly fit.
